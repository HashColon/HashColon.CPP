# HashColon Library
C++ utility library by HashColon(wchyoo@snu.ac.kr)

## Getting started 
* Prerequisites
    - Boost C++ libraries: 
      (https://www.boost.org/)
    - RapidJSON: A fast JSON/Parser for C++
      (https://rapidjson.org/)  
    - Eigen: C++ template library for linear algebra
      (https://eigen.tuxfamily.org/index.php?title=Main_Page)
